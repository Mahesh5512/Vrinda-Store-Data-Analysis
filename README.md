# Vrinda-Store-Data-Analysis
Project Overview

This project is an interactive Excel Sales Dashboard created to analyze Vrinda Storesâ€™ annual sales performance for the year 2022. The dashboard provides a clear and visual summary of sales trends, customer behavior, order status, and channel-wise performance to support business decision-making.

The dashboard is fully interactive using Excel slicers, charts, and formulas, allowing users to filter data by month, sales channel, and product category.

- Objectives

Analyze monthly orders vs sales performance

Compare men vs women sales contribution

Track order status (Delivered, Cancelled, Refunded, Returned)

Identify state-wise sales performance

Understand age group and gender-based buying patterns

Analyze sales contribution by online channels
